## 3. Key Concepts (Explained Gently)

### a) Sidebar Menu

In Streamlit, we use:

```python
st.sidebar
```

to place widgets (like selectbox, radio) on the **left side**.

Example:

```python
page = st.sidebar.radio("Go to", ["Home", "Dashboard", "Profile"])
```

This gives a simple sidebar menu.

---

### b) Top Navigation Bar (Tabs)

Streamlit has:

```python
st.tabs(["Tab 1", "Tab 2"])
```

Each tab can show different content.

---

### c) Changing Visible Content

We use **if / elif** in Python to decide what to show:

```python
if page == "Home":
    # show home content
elif page == "Dashboard":
    # show dashboard content
```

Inside dashboard:

```python
tab1, tab2 = st.tabs(["Today", "This Week"])
with tab1:
    # content for today
with tab2:
    # content for this week
```
