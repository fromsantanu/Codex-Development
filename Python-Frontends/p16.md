## **1. Simple Introduction**

A **dashboard** is just a screen that shows the most important information in one place.
Think of it like opening a single page and immediately understanding:

* What is happening?
* What needs attention?
* What actions you may need to take?

Dashboards help users avoid searching through many pages. Everything they need is visible at a glance.

## **5. Example UI Code (Using Streamlit)**

Here is a small, clean dashboard using **Streamlit**:

```python
import streamlit as st
import pandas as pd
import altair as alt

st.title("Simple Clinic Dashboard")

# --- Filters ---
date_filter = st.date_input("Choose a date")

# --- Cards ---
col1, col2, col3 = st.columns(3)
col1.metric("Patients Today", 18)
col2.metric("New Registrations", 4)
col3.metric("Pending Follow-ups", 6)

# --- Table ---
data = {
    "Name": ["Rita", "Amit", "John"],
    "Status": ["Waiting", "In Progress", "Completed"],
    "Time": ["10:30", "11:00", "11:30"]
}
df = pd.DataFrame(data)
st.subheader("Patient List")
st.table(df)

# --- Chart ---
chart_data = pd.DataFrame({
    'day': ['Mon', 'Tue', 'Wed', 'Thu', 'Fri'],
    'patients': [12, 18, 20, 15, 22]
})
chart = alt.Chart(chart_data).mark_bar().encode(
    x='day',
    y='patients'
)
st.subheader("Patients Over the Week")
st.altair_chart(chart, use_container_width=True)
```

This shows:

* cards
* table
* chart
* date filter

Simple, clean, easy for beginners.

## **6. Backend API Example (Simple)**

Your frontend reads this JSON and fills the cards, tables, and charts.
