2. Paginate Pagination results
   - Explanation Pagination feels like keeping patient notes on neat pages so the desk does not get overwhelmed.
   ```jsx
   function PaginationPagination() {
     const [page, setPage] = useState(1);
     return (
       <div>
         <p>Page {page} of Pagination records.</p>
         <button onClick={() => setPage(p => Math.max(1, p - 1))}>Previous</button>
         <button onClick={() => setPage(p => p + 1)}>Next</button>
       </div>
     );
   }
   ```
   - Codex help: Request Codex to add pagination controls for Pagination so staff can flip through pages easily.

3. Call the API for Pagination
   - Explanation Connect to /api/appointments with GET so the sunrise clinic frontend keeps Pagination mirrored with the backend.
   ```jsx
   const [records, setRecords] = useState([]);
   useEffect(() => {
     async function syncData() {
       const response = await fetch('/api/appointments', {
         method: 'GET',
       });
       if (!response.ok) throw new Error('Failed to load Pagination');
       const data = await response.json();
       setRecords(data);
     }
     syncData();
   }, []);
   return <p>Pagination records: {records.length}</p>;
   ```
   - Codex help: Ask Codex to craft the fetch block that calls /api/appointments for Pagination and stores the result.
