# Chapter 12. Styling UI

## Introduction
- The Styling UI page helps the Sunrise Clinic Scheduling Desk team handle the clinic flow without confusion.
- The React front end calls /api/appointments so this screen stays in sync with the backend API.

## Key Concepts
- Buttons and inputs are the buzzers staff tap while telling a patient where to check in.
- Styling UI stays relatable when each click feels as calm as filling out a check-in slip at Sunrise Clinic Scheduling Desk.

## Step-by-step Implementation
1. Lay out the Styling UI intro card
   - Explanation Start with a friendly section so the Sunrise Clinic desk knows what Styling UI is about.
   ```jsx
   function StylingUiIntro() {
     return (
       <section className='intro-card'>
         <h2>Styling UI at Sunrise Clinic Scheduling Desk</h2>
         <p>Welcome back, front desk! This section explains why Styling UI matters today.</p>
       </section>
     );
   }
   ```
   - Codex help: Ask Codex to build the StylingUiIntro section with a warm heading and short explanation for Styling UI.

2. Handle Styling UI interactions
   - Explanation Add a button or input so Styling UI responds when the receptionist taps or types.
   ```jsx
   function StylingUiButton() {
     const [state, setState] = useState('ready');
     return (
       <button onClick={() => setState(state === 'ready' ? 'sent' : 'ready')}>
         Styling UI: {state}
       </button>
     );
   }
   ```
   - Codex help: Prompt Codex to add a Styling UI button that toggles its label whenever the receptionist clicks it.

3. Call the API for Styling UI
   - Explanation Connect to /api/appointments with GET so the sunrise clinic frontend keeps Styling UI mirrored with the backend.
   ```jsx
   const [records, setRecords] = useState([]);
   useEffect(() => {
     async function syncData() {
       const response = await fetch('/api/appointments', {
         method: 'GET',
       });
       if (!response.ok) throw new Error('Failed to load Styling UI');
       const data = await response.json();
       setRecords(data);
     }
     syncData();
   }, []);
   return <p>Styling UI records: {records.length}</p>;
   ```
   - Codex help: Ask Codex to craft the fetch block that calls /api/appointments for Styling UI and stores the result.
