3. Call the API for Clinic Appointment UI
   - Explanation Connect to /api/clinic/appointments with GET so the sunrise clinic frontend keeps Clinic Appointment UI mirrored with the backend.
   ```jsx
   const [records, setRecords] = useState([]);
   useEffect(() => {
     async function syncData() {
       const response = await fetch('/api/clinic/appointments', {
         method: 'GET',
       });
       if (!response.ok) throw new Error('Failed to load Clinic Appointment UI');
       const data = await response.json();
       setRecords(data);
     }
     syncData();
   }, []);
   return <p>Clinic Appointment UI records: {records.length}</p>;
   ```
   - Codex help: Ask Codex to craft the fetch block that calls /api/clinic/appointments for Clinic Appointment UI and stores the result.
